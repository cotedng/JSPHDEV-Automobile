There are two project folders
The server project remains the same as the last unit
The client project contains a terminal UI client and a web UI client running on Tomcat
In order to test the project, firstly you need to launch the server and use the terminal UI client(Client.class) to upload properties files into the server.
Once the server obtain automobiles, you can start Tomcat and go to http://localhost:8080 and select an automobile to configure.
